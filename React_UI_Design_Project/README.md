A modern React UI project built using React, Vite, Tailwind CSS, and reusable component architecture.

## 🚀 Features

- Reusable React Components
- Tailwind CSS Styling
- Clean Folder Structure
- Fast Development with Vite
- Scrolling feature within cards

## 🛠️ Tech Stack

- React.js
- Vite
- Tailwind CSS
- JavaScript
- CSS3
